# Setup - Boutique Spark

## Installation

```bash
npm install
```

## Développement

```bash
npm run dev
```

Le serveur démarre sur `http://localhost:8080`

## Build

```bash
npm run build
```

Les fichiers compilés se trouvent dans le dossier `dist/`

## Tests

```bash
# Une seule exécution
npm run test

# Mode watch
npm run test:watch
```

## Linting

```bash
npm run lint
```

## Variables d'Environnement

Créer un fichier `.env.local` si nécessaire pour les variables d'environnement.
